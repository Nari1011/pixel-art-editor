# Pixel Art Editor

A browser-based pixel art editor built with React + TypeScript + Vite.

## Features

- **Drawing tools**: Pen, Eraser, Flood Fill, Eyedropper
- **Color picker** with a 20-color preset palette
- **Canvas sizes**: 16×16, 32×32, 64×64
- **Undo / Redo** (up to 50 steps, Ctrl+Z / Ctrl+Y)
- **Grid toggle**
- **Export to PNG**
- Touch support

## Tech Stack

- React 19
- TypeScript
- Vite

## Getting Started

```bash
npm install
npm run dev
```

Open http://localhost:5173
